

# How US College Students Use AI in 2025: A Quantitative Snapshot

This project presents a secondary analysis exploring how college students in the United States are engaging with artificial intelligence tools in 2025. Using data sourced from Reddit discussions, statistical databases like Pew Research and Statista, and public reports, this study visualizes behaviors, sentiment trends, and usage patterns.

## 🔍 Focus & Methodology
- 📚 Secondary data analysis from digital forums and research databases
- 💬 Thematic exploration based on Reddit threads and survey datasets
- 📊 Descriptive visualization using Python


## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/mavenpro123/USCollegeStats.git`
2. Create a virtual environment and install dependencies:  
   `pip install -r requirements.txt`
3. Explore the Jupyter notebooks to dive into the visual findings.

## 💡 Key Themes Explored
- Trends in AI tool preferences among students
- Discussions around ethics, productivity, and academic integrity
- Data-based storytelling with charts and plots
